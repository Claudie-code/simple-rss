# SimpleRSS

A minimalist RSS reader built with React and Next.js.

## Demo

[View the live site](https://www.simple-rss.com)

## Features

- Fetch and parse RSS feeds from any URL
- Minimalist, responsive design
- Supports multiple feeds

## Tech Stack

- **React 18**
- **Next.js**
- **TailwindCSS**
- **Radix UI**
- **Supabase** (for backend/data management)
- **RSS Parser & RSS Finder** (feed parsing)
- **TypeScript**
- **AOS** (animations)
- **Highlight.js & Prism.js** (syntax highlighting)

## Installation

```bash
git clone https://github.com/Claudie-code/simple-rss.git
cd simple-rss
npm install
npm run dev
```
